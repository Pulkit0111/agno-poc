# Quick Notes

Use this skill when the user asks to create a simple notes file in the workspace and verify it by reading it back.

1. Create or overwrite `notes.txt` in the workspace.
2. Put the requested notes in the file, usually as a short bullet list.
3. Read the file back to confirm the contents.

Keep the workflow short and direct.
