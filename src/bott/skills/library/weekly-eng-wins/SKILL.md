# Weekly Eng Wins

Use this workflow when asked to summarize recent activity in a Slack channel as a weekly "wins" blurb for a newsletter.

## Goal
Produce 3 concise win items based on recent channel interactions. Each item should be upbeat, specific, and written as exactly 2 sentences.

## Process
1. Read the recent channel history.
2. Select 3 recent interactions that are:
   - substantive
   - positive or progress-oriented
   - distinct from one another
3. For each selected interaction:
   - infer the underlying win, not just the literal message
   - write 2 upbeat sentences
   - keep the tone warm, precise, and newsletter-friendly
4. Prefer concrete outcomes, shipped work, demos, unblockings, or capability expansions.
5. Avoid restating raw message text verbatim unless it is already polished.
6. Do not mention that the items came from Slack history unless asked.

## Writing style
- Warm, polished, and concise.
- Mildly celebratory, but not overly cheerful.
- Focus on impact and momentum.
- Keep it suitable for an internal newsletter.

## Output shape
- Provide a short intro if helpful.
- Then list 3 win blurbs.
- Each blurbs should be 2 sentences.
- Avoid bullets with long explanations; keep it readable and ready to paste.
