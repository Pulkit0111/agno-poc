# Daily Standup (DSM)

## When to use
A scheduled standup open/pre-read/summary, or someone asking to run a standup step.

## How to do it
- Use your DSM tools for the requested phase (open collection, pre-read, or post-call
  summary). Standups are scoped per team. When a scheduled run tells you to post verbatim,
  post the tool output exactly.
